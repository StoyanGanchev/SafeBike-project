Hardware

The SafeBike project contains hardware managed by software written on ArduinoIDE. The hardware is made from an arduino microcontroller and sensors connected with it. The Arduino board sends and receives signals (analog or digital) to the sensors. Depending on the returned information, outputs are controlled (lcd display, LED, etc.). I use ArduinoNANO to control the sensors, as it is small in size but contains the necessary inputs and outputs. The power supply is 5V and can be connected directly to the sensors.

I made a real model in order to visualize the project.
- To detect distance, I use ultrasonic sensor HC-SR04. It sends ultrasonic waves and detects a distance relative to the time of their return.
- To display information while cycling, I use an lcd display.
- To detect the slope of the road, I use a sensor (gyroscope) GY-521.
- To read the turning angle, I use a potentiometer.
- To read the rpm of the wheel, I use an infrared diffusion sensor. The light is emitted by the sensor, part of the light reflected by the object is returned to the sensor and a pulse is detected.
